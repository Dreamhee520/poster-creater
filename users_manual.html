获奖名单系统
使用说明

本指南将帮助您快速上手使用我们的获奖名单管理与展示系统， 轻松实现数据导入、样式定制和成果展示。

开始探索

快速开始

步骤 1: 准备 Excel 数据

创建一个 Excel 文件，包含至少 姓名、学院 和 作品名称 三列。 您还可以添加第四列 奖次 来区分不同奖项。第一行请作为表头。

步骤 2: 使用 Excel 转 JSON 工具

打开 excel_to_json.html 文件，上传您的 Excel 文件。 在列映射界面，将 Excel 列与系统字段对应，然后点击"生成 JSON"并下载。

步骤 3: 在主系统中加载数据

打开系统主页面 index.html，点击"读取数据"按钮， 选择您刚刚生成的 JSON 文件。系统将自动加载并展示数据。

设计初衷

为什么创建这个系统？

这个项目是基于纯 H5 实现的，旨在提供一个方便实用的获奖名单展示解决方案。

最初设计这个软件的灵感来源于一个实际需求：我需要为学校制作一份获奖名单海报， 但在网上翻遍了各种海报设计软件，要么功能复杂难以操作，要么不符合我的需求。 手动制作海报又耗时费力，效率低下。

于是，我决定开发一个专门用于展示获奖名单的海报生成器，既能够快速导入数据， 又能够灵活定制样式，让用户能够轻松创建专业美观的获奖名单展示页面。

关于未来更新：由于这是一个个人项目，未来大概率不会进行大规模更新。 不过，这个系统的设计具有很好的扩展性，如果有开发者愿意贡献更多的模板或功能， 可以直接进行添加和扩展。

常见问题

导入 Excel 时，列映射是什么意思？

列映射是指将您 Excel 文件中的列（例如 "学生姓名"）与系统期望的字段（例如 "姓名"）进行关联。 这一步确保系统能正确识别和读取您的数据。如果您的 Excel 表头是标准的 "姓名", "学院", "作品名称", "奖次"， 系统通常能自动匹配。

为什么我导入后看不到数据？

请检查以下几点：

  1. 确保您下载的是生成的 .json 文件，而不是原始的 .xlsx 文件。
  2. 在主系统中，点击"读取数据"后，选择的是正确的 JSON 文件。
  3. 检查您的 Excel 文件格式是否正确，特别是表头是否存在。

我可以在手机上使用这个系统吗？

可以。系统采用响应式设计，在手机、平板等移动设备上都能良好工作。 不过，由于屏幕尺寸限制，某些操作（如精细调整样式）在桌面端会更加方便。

我的数据保存在哪里？安全吗？

所有数据都保存在您本地浏览器的 localStorage 中。 这意味着：

  1. 数据完全保存在您自己的设备上，非常安全。
  2. 清除浏览器缓存或更换浏览器/设备后，数据会丢失。
  3. 建议定期使用"保存数据"功能导出 JSON 文件进行备份。
